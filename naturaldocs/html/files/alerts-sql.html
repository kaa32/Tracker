insert_new_alert(int, int, text, json, json)

CREATE OR REPLACE FUNCTION msc.insert_new_alert(
   uid int,
   alert_typ int,
   alert_txt text,
   alert_src json,
   alert_trg json DEFAULT NULL
) RETURNS BIGINT

Inserts new alert into table msc.alerts.  Function checks user tariff-mask for ability to send messages.  This function is recommended for creating user-to-user messages

Parameters

uid (int)identifier of user that alert is appointed to
atype (int)type of alert.  Magic number for web coder purposes
atext (text)text of alert that should be converted to displayed text
src (json)json-container for alert source description.
trg (json)json-container for alert target description.  DEFAULT NULL.  Optional field used for alerts that have a target

Returns

BIGINTidentifier of alert (or 0 for fail case)

See also

msc.alerts.  JSON container format

msc.constants

constants.sql

Example

select msc.insert_new_alert(uid, alert_typ, alert_txt, alert_src, alert_trg);

generate_alert(int, text, text, int, text, text, int)

CREATE OR REPLACE FUNCTION msc.generate_alert(
   uid int,
   alert_txt text,
   src_object text,
   src_name text,
   src_id int,
   trg_object text DEFAULT NULL,
   trg_name text DEFAULT NULL,
   trg_id int DEFAULT NULL
) RETURNS BIGINT

Inserts new alert into table msc.alerts.  This is an internal function for automatic alert generating functional.  Function generates alerts and ignores MESSAGE_MASK for user.

Parameters

uid (int)identifier of user that alert is appointed to
alert_txt (text)text of alert that should be converted to displayed text
src_object (text)type of object-source of alert
src_name (text)name of object-source of alert
src_id (int)identifier of object-source of alert
trg_object (text)type of object-target of alert, DEFAULT NULL,
trg_name (text)name of object-target of alert, DEFAULT NULL,
trg_id (int)identifier of object-target of alert, DEFAULT NULL

Returns

BIGINTidentifier of new alert (or 0 for fail case)

Example

select msc.generate_alert(1, ‘USER ADDED TO CIRCLE’, ‘user’, ‘andrew’, 2, ‘circle’,’albert circle’, 1);
