insert_new_billrec(int, int, text)

CREATE OR REPLACE FUNCTION msc.insert_new_billrec (
   uid INT,
   tid INT,
   tariff_start TEXT DEFAULT NULL
) RETURNS INTEGER

Inserts new billing record into table msc.billrecs.  Generates alert BILLREC CREATED on successful return

Parameters

uid (int)identifier of user owned this billing record
tid (int)identifier of tariff for billing record
tariff_start (text)text representation of time at time zone ‘UTC’(!) where tariff should start.  Recommended format ‘YYYY-MM-DD HH:MI:SS’.  Time of day is optional.

Returns

INTEGERidentifier of newly created billrec (or 0 for fail case)

Examples

select msc.insert_new_billrec(1, 1);

select msc.insert_new_billrec(2, 1, ‘2014-01-01’);

select msc.insert_new_billrec(10, 2, ‘2013-12-31 23:15:00’);

CREATE OR REPLACE FUNCTION msc.update_user_billrec(uid int) RETURNS INTEGER

Updates user billing records up to his/her relevant record for current time.  Raises exception if user has no free tariff (security incident).  Mostly used by API-function msc.update_all_users_billrecs()
