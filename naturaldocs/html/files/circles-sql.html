insert_new_circle(int, text, text, boolean)

CREATE OR REPLACE FUNCTION msc.insert_new_circle(
   uid int,
   cname text,
   cdesc text DEFAULT NULL,
   is_public boolean DEFAULT FALSE
) RETURNS INTEGER

Inserts new circle into table msc.circle.  Generates alert CIRCLE CREATED on successful return.  After circle creation the user becomes the circle member.

Parameters

uid (int)identifier of user that creates circle
cname (text)name of circle, unique per user
cdesc (text)circle description, DEFAULT NULL
is_public (bool)optional status that circle is public, DEFAULT FALSE

At this moment property is_public does not processed.

Returns

INTEGERidentifier of newly created circle (or 0 for fail case)

Examples

select msc.insert_new_circle(1, ‘circle-name’, ‘circle-description’);

CREATE OR REPLACE FUNCTION msc.delete_circle(cid int) RETURNS INTEGER

Deletes circle.  Before deletion all members are removed from circle automatically.  Alert CIRCLE DELETED is generated for circle master.

Recommended but not required that only the circle master can delete circle.

Parameters

cid (int)circle identifier

Returns

CREATE OR REPLACE FUNCTION msc.remove_user_from_circle(uid int,

Removes user from circle.  Circle master cannot be removed from his circle - in that case function does nothing.  Alerts USER REMOVED FROM CIRCLE are generated by trigger for ALL members of circle.
