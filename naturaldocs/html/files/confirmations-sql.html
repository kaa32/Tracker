insert_new_confirmation(int, text, text, text, text)

CREATE OR REPLACE FUNCTION msc.insert_new_confirmation(
   uid int,
   token text,
   act_name text,
   act_params text,
   alert_type int,
   alert_src json,
   alert_trg json,
   confirm_text text
) RETURNS BIGINT

Inserts new confirmation into table msc.confirmations.

Checks if act_name has corresponding name in table msc.actions.  If act_name not found then function returns NULL and exits.  Generates alert of type 2

Parameters

uid (int)identifier of user that confirmation appointed to
token (text)confirmation token, UNIQUE
act_name (text)name of action
act_params (text)parameters for actions.  Please, be very careful, cause it can not be verified until the real function call.
confirm_text (text)text that to be converted to displayed text of confirmation.

Returns

INTEGERidentifier of newly created confirmation (or NULL for fail case)

CIRCLE_DELETE_ACTIONcircle_id(int)
GEOZONE_CREATE_ACTIONuser_id(int), circle_id(int), zname(text), zdesc(text), center_lat(float), center_lon(float), radius(float)
GEOZONE_DELETE_ACTIONgeozone_id(int)
USER_TO_CIRCLE_ADD_ACTIONuser_id(int), circle_id(int)
USER_FROM_CIRCLE_REMOVE_ACTIONuser_id(int), circle_id(int)
USER_TO_GEOZONE_ADD_ACTIONuser_id(int), geozone_id(int)
USER_FROM_GEOZONE_REMOVE_ACTIONuser_id(int), geozone_id(int)
USER_TO_GEOZONE_CONNECT_ACTIONuser_id(int), geozone_id(int)
USER_FROM_GEOZONE_DISCONNECT_ACTiONuser_id(int), geozone_id(int)

Examples

select msc.insert_new_confirmation(17, ‘token5’, ‘CIRCLE_DELETE_ACTION’, ‘17’, ‘Remove circle?’);

select msc.insert_new_confirmation(1, ‘token6’, ‘GEOZONE_CREATE_ACTION’, ‘1, 21,’’zone1’’, ‘’desc1’’, 0, 0, 200’, ‘Create geozone?’);
