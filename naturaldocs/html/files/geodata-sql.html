insert_new_geodata(int, float, float)

CREATE OR REPLACE FUNCTION msc.insert_new_geodata(did int,
lat float,
lon float) RETURNS BIGINT

Inserts into table msc.geodata new geodata from device by device identifier.  Overloaded.  After inserting it updates field ‘device_current_geodata’ in table msc.devices and calls the functions which forming geozone alerts for geozone consumers.  Generates JSON-fields from parameters and save it in table msc.geodata

Parameters

did (int)identifier of device
lat (float)geodata latitude.  Minus for south latitude.
lon (float)geodata longitude.  Minus for west longitude.

Returns

BIGINTnewly inserted geodata identifier, 0 if fail

See also

msc.geodata.  JSON container format

<generate_alerts_fgid_mid(bigint, int)>

Examples

select msc.insert_new_geodata(2, 12.345678, -9.876543);
