insert_new_tariff(text, text, int, int, text, boolean)

CREATE OR REPLACE FUNCTION msc.insert_new_tariff(
   tname text,
   tdesc text,
   ttop int,
   tmask int,
   tinterval text,
   tfree boolean default FALSE,
   tshowcase boolean default FALSE,
   tprice float default 0.0
) RETURNS INTEGER

Inserts new tariff into table msc.tariffs.  Requires admin privileges to do that

Parameters

tname (text)name of new tariff, UNIQUE
tdesc (text)description of new tariff
ttop (int)tariff limitations value.
tmask (int)tariff mask value
tinterval (text)text representation of tariff duration interval
tfree (boolean)free-status of tariff, DEFAULT FALSE Recommended format of tariff interval ‘NN days’.

Returns

INTEGERidentifier of newly created tariff (or 0 for fail case)

See also

msc.constants

constants.sql

Examples

select msc.insert_new_tariff(‘free-month’, ‘free for 1 month’, 10000, ‘31 days’, ‘t’, ‘t’, 0);

select msc.insert_new_tariff(‘standard-10’, ‘standard tariff’, 10, ‘1 month’);

select msc.insert_new_tariff(‘super-unlimited’, ‘’, 10000, ‘100 years’);

Sets tariff price value by tariff identifier.  If newprice is less than 0, or tariff_is_free is set to ‘TRUE’, then function does nothing and return 0
