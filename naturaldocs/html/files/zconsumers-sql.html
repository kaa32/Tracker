connect_user_to_geozone(int, int)

CREATE OR REPLACE FUNCTION msc.connect_user_to_geozone(uid int,
zid int) RETURNS INTEGER

Inserts new zconsumer into into table msc.zconsumers.  Function checks if geozone-master reached tariff limitations.  If she reached then function generates alert LIMITS REACHED and return NULL.  Invited user and geozone should have one circle - it is verified inside function.  On successful insert function generates alert USER CONNECTED TO GEOZONE for geozone master and new consumer

Parameters

uid (int)invited user identifier
zid (int)geozone identifier

Returns

INTEGERidentifier of newly created zconsumer (or 0 for fail case)

Examples

select msc.connect_user_to_geozone(2, 1);
