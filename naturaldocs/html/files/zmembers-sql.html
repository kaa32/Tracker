add_user_to_geozone(int, int)

CREATE OR REPLACE FUNCTION msc.add_user_to_geozone(uid int,
zid int) RETURNS INTEGER

Inserts new zmember into into table msc.zmembers.  Invited user and geozone should have one circle - it is verified inside function.  Generates alert USER ADDED TO GEOZONE for all consumers of that geozone.

Parameters

uid (int)invited user identifier
zid (int)geozone identifier

Returns

INTEGERidentifier of newly created zmember (or 0 for fail case)

Examples

select msc.add_user_to_geozone(2, 1);

get_member_alert_count(int, int)

CREATE OR REPLACE FUNCTION msc.get_member_alert_count(uid int,
zid int) RETURNS INTEGER

Returns remaining alert_count for user-member by user-member identifier and geozone identifier.  Alert count decrements on every geozone alert (such as USER INSIDE/OUTSIDE/TOGGLES GEOZONE).  If alert count <= 0 then alerts do not generated more.

Parameters

uid (int)user identifier
zid (int)geozone identifier

Returns

INTEGERalert_count for that zmember

Examples

select msc.get_member_alert_count(1, 1);
